Why I built this

It is March 2026. No one should be spending their time hand-making slide decks anymore. I built this for myself and other Claude Code maximalists who have fully converted to creating slides in HTML.

When you open an HTML file in a browser tab, there's no easy way to make it go fullscreen. And when you connect to a bigger projector, the zoom level is off — and you're fumbling with it mid-presentation. That's what this fixes.

I tried the existing AI slide tools. They don't really work — another system to learn, another subscription, no continuity with the context Claude already has about my product and brand. I also tried having Claude write PowerPoint files directly. Those don't come out well either.

What works: having Claude Code write HTML. The output is consistently more precise and more beautiful than anything else I've tried.
